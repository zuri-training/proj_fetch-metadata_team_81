What is a Metadata?

Metadata summarizes basic information about data, which can make it easier to find, use and reuse particular instances of data. For example, author, date created, date modified and file size are examples of very basic document file metadata. Having the ability to search for a particular element (or elements) of that metadata makes it much easier for someone to locate a specific document.
